Dax Demo #01

You should see below a white triangle and a transparent blue square. Show Stats.

This demo showcases :

  • Models with simple meshes and default materials
  • The PositionLayer shader
  • The ColorLayer shader
  • The (dumb) material shader preprocessor
  • Rotations of models
        
          World world = new World();

          Model triangle = new SingleTriangleModel();
          world.add(triangle);

          Model square = new UnitSquareModel();
          world.add(square);

          square.material.setLayerVariable('ColorLayer', 'uColor', [0.0, 0.0, 1.0]);
          square.material.setLayerVariable('ColorLayer', 'uAlpha', 0.5);
